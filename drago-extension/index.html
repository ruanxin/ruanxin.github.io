About drago


Drag to go is not a new thing, it’s also my favorite feature while using browser, so extension like Easy DragToGo on Firefox, yet Another Drag and Go on Chrome is always my first choice after installed. Unfortunately I can’t find a similar extension exists on Safari:( Since nothing exists, why not just create it, that’s the plan to make drago. The first version I only implement the basic feature to make link, text or image are draggable and with different behaviors. If you have any good opinion or feedback on this extension, feel free to leaving comments or contact me.

Change log

[1.0] 2015-11-17

  • initial release
  • [new] Drag link to open in a new tab.
  • [new] Drag text to search by user-defined search engine in a new tab.
  • [new] Drag image to open in a new tab or find similar images by TinEye.
  • [new] user-defined new tab mode - foreground or background.
  • [new] user-defined new tab position.
