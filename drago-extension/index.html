About drago


Drago is a drag to go extension for Safari. This feature is not a new thing, it’s also my favorite extension while using browser, like Easy DragToGo on Firefox, Yet Another Drag and Go on Chrome. Unfortunately I can’t find a similar extension exists on Safari:( Since nothing exists, I’m thinking why not just create it, that’s the plan to make drago. The first version I only implement the basic feature to make link, text or image are draggable and with different behaviors. If you have any good opinion or feedback on this extension, feel free to leaving comments or contact me.

Change log

[1.1] 2016-02-18

  • [new] New tab can be opened nearby current tab.
  • [new] New tab can be opened depending on mouse drag direction.

[1.0] 2015-11-17

  • initial release
  • [new] Drag link to open in a new tab.
  • [new] Drag text to search by user-defined search engine in a new tab.
  • [new] Drag image to open in a new tab or find similar images by TinEye.
  • [new] User-defined new tab mode - foreground or background.
  • [new] User-defined new tab position.
